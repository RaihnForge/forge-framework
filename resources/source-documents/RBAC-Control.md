### Role-Based Access Control Matrix (RBAC)
Fourth State Communications ensures that privileged permissions are segregated from non-privileged personnel and accounts. Administrative access to production infrastructure strictly adheres to the principle of least privilege. User access to systems is provisioned based on the principle of least privilege.

Fourth State Communications conducts quarterly reviews of user access to production servers, databases, and sensitive applications to validate that internal user access, service account usage, and shared account access aligns with the principle of least privilege. Following each access review, the quarterly review template is updated with the completion date. Any identified access issues are documented and tracked for remediation.

It is required that an auditor, distinct from the reviewer, verifies the access rights since individuals cannot audit their own access.
