# The Forge Framework

A theory of organizational management for individuals and organizations.

> You have all the tools you ever need within yourself to become the person you wish or build the organization you wish — both oriented to changing the world in a positive way within the constraints of our ephemeralness.

## Three-Tier Documentation Strategy

1. **Master Reference** (`docs/`) — The extended outline and core concepts. Source of truth. Always the most complete and current.
2. **Evidence Collection** (`evidence/`) — Stories, experiences, proofs, studies. Raw material that brings concepts to life and entertains the reader.
3. **Published Guide** (`guide/`) — The polished, reader-facing publication. Draws from both. Structured for clarity, enriched with evidence, edited for impact.

The Master Reference controls the Guide. The Evidence Collection feeds the Guide.
